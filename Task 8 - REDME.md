# Task 8 - Simple Sales Dashboard

## Objective
To build an interactive Sales Dashboard using Power BI that analyzes sales performance by month, region, and category.

## Dashboard Features
- Sales Over Months (Line Chart)
- Sales by Region (Bar Chart)
- Sales by Category (Donut Chart)
- Region Slicer for interactive filtering

## Tools Used
- Microsoft Power BI
- Superstore Sales Dataset

## Outcome
Developed an interactive dashboard to visualize sales trends, regional performance, and category-wise contribution, enabling better business insights.
